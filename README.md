# Сервис поиска свободной аудитории на МатМехе

**Техническое задание**

**Цель проекта:**  
Разработать web-приложение для студентов и преподавателей для поиска и бронирования аудиторий на МатМех УРФУ

**Описание проекта:**

Сейчас на матмехе часто аудитории оказываются заняты, поэтому преподавателям и студентам приходится искать класс, где они могли бы провести пару. Наш сервис позволит бронировать аудитории и наглядно видеть какие аудитории заняты и когда они освободятся.

**Требования к интерфейсу:**

- ПК и мобильная версия интерфейса  
- Вид сверху на 5-6 этажи для наглядности. Занятая аудитория закрашена красным, свободная \- зеленым.   
- Фильтры по времени, дате, этажу, наличию проектора, (количество окон?)  
- Отдельный интерфейс для мобилки: придумать 

**Дополнительные требования:**

- Приложение должно поддерживать аутентификацию сотрудников
